    Type-Casting and subtype⚓︎

    Conversion of one data type to another is called type-casting.

    Two type of type-casting: 1. Implicit Type Conversion: Where Python itself convert one to another datatype.

    # implicit type conversion
a = 23
b = 23.4

print(type(a))
print(type(b))
    1. Explicit Type Conversion: Where the user converts one datatype to another.
    # Explicit Type Conversion
a = '200'
print(type(a))

b = int(a)
print(type(b))  # string to int

    Operators and Operands⚓︎

    Operators indicates what operations is to be performed. Operands indicates on what the action or the operation should be performed.

    x + y = 10, here x, y and 10 are operands and + is operators

    type of operators⚓︎

    python divides the operators in the following groups:

    • Arithmetic operators
    # Arithmetic Operators
a = 10
b = 3

print(a + b) # Addition
print(a - b) # Subtraction
print(a * b) # Multiplication
print(a / b) # Division
print(a % b) # Modulus
print(a ** b) # Exponentiation

# Note that if one of the values is a float, you'll get back a float.
print(a // b) # Floor division
    • Assignment operators
    # Assignment Operators
a = 10
b = 3

a += b # a = a + b
print(a)

a -= b # a = a - b
print(a)

a *= b # a = a * b
print(a)

a /= b # a = a / b
print(a)
    • Comparison operators
    # Comparison Operators
a = 10
b = 3

print(a == b) # Equal to
print(a != b) # Not equal to
print(a > b) # Greater than
print(a < b) # Less than

    The == operator (equal to) asks whether two values are the same as each other. The = operator (assignment) puts the value on the right into the variable on the left.

    • Logical operators
    # Logical Operators
a = True
b = False

print(a and b) # and - both conditions must be true
print(a or b) # or - at least one conditions must be true
print(not a) # not - inverts the condition
    • Identity operators
    # Indentity Operators- used to check the indentity
a = 10
b = 3

print(a is b) # is
print(a is not b) # is not
    • Membership operators
    # Membership Operators- used to check the presence of a sequance in an object
a = "Hello"
b = 'e'

print(b in a) # in
print(b not in a) # not in
    • Bitwise operators
    # Bitwise Operators - difficult
a = 10
b = 2

print(a & b) # and
print(a | b) # or
print(a ^ b) # xor

print(~a) # not
print(a << b) # left shift - Shifts the bits of the number to the left by the number of bits specified.
print(a >> b) # right shift - same do for right

    shortcut math expression

     a = 2

    Conditional Statements⚓︎

    Conditional statements allows computer to execute a certain condition only if it is true.

    Type of conditional statements⚓︎

    1. if
    2. if-else
    3. if-elif-else
    4. Nested statement
    # if
marks = 82

if marks >= 90:
    print("Grade A")

print("Other Rank")

    # if-else
age = 20
if age >= 18:
    print("You are eligible to vote")
else:
    print("You are not eligible to vote")

    # if-elif-else
marks = 76

if marks >= 90:
    print("Grade A")
elif marks >= 80:
    print("Grade B")
elif marks >= 70:
    print("Grade C")
else:
    print("Grade D")

    # nested statement
num = 10

if num >= 0:
    if num == 0:
        print("Zero")
    else:
        print("Positive number")
else:
    print("Negative number")

    # short hand if statement - one line statement
a = 10
b = 20

if a < b: print("a is less than b")

    short hand if-else statement⚓︎

    Conditional expression - A one line shortcut for the if-else statement(Ternary Operator). here, X if condition else Y

    a = 10
b = 20

# body of if; if condition; else; body of else
print("a is less than b") if a < b else print("a is greater than b")

    Loops⚓︎

    A loop means to repeat something in the exact same way.

    types of loops are:⚓︎

    1. For
    2. While
    3. While True
    4. Nested
    # for loop
for i in range(10): # range() means range between --> 1 to 10
    print(i)

    # while loop
i = 0 # start
while i < 10: # condition
    print(i)
    i += 1 # increment

    # while true -- not important, rarely use case
i = 0
while True:
    print(i)
    i += 1
    if i >= 100:
        break

    # nested loop - a loop inside a loop is called as nested loop.
for i in range(1,6):
    for j in range(1, i+1):
        print(j, end = ' ')
    print()

    # for loop with conditional statement
for i in range(1,10):
    if i % 2 == 0:
        print(i)

# conditional statement with and operator
for i in range(1, 101):
    if i % 2 == 0 and i % 3 == 0:
        print(i)

    num = [2,2,3,1,3,4,6]
# enumerate return value and index
for index, value in enumerate(num):
    print(index,value)

    Break and Continue Statement⚓︎

    Continue Statement⚓︎

    Continue statement is used when you want to skip a particular condition.

    # Continue statement
for i in range(1, 10):
    if i == 5:
        continue
    print(i)

    Break Statement⚓︎

    Break statement is used when you want to destroy a loop at a certain condition and come out of the loop.

    # Break statement
for i in range(1, 10):
    if i == 5:
        break
    print(i)
